# Chrome Profile Switcher

Chrome Profile Switcher is a lightweight Windows utility for quickly launching and switching between Google Chrome profiles from a single interface. It runs in the system tray and restores its window when reopened, rather than launching multiple instances.

The project is source-visible but **not open source**.

---

## Features

- One-click launching of Chrome profiles
- Native Windows system tray integration
- Restores existing instance when reopened
- Uses Chrome’s native profile directories
- Portable single-EXE build
- No telemetry or network access

---

## Requirements

- Windows 10 or Windows 11
- Google Chrome installed

---

## Installation

Download the latest executable from the **Releases** section and run it.
No installer is required.
